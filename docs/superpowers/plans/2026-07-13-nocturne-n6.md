# Nocturne N6 — Ops create flows + resilience surfaces — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Add CREATE flows to the operations console for the three Phase-3 engines (rotation policies, sync targets, dynamic roles) via kit `Sheet` forms, removing the "use the CLI" empty-state copy; and add an app-level `ErrorBoundary` + a real 404 route replacing the silent redirect.

**Architecture:** Each ops panel (`RotationPanel`/`SyncPanel`/`DynamicPanel`) gains a "New …" button opening a kit `Sheet` create form. All three creates need a target config, so a shared `ConfigPicker` (fed by the existing 403-tolerant `useProjectConfigMap`) is built once in T1 and reused. Secrets-bearing fields (DSNs, PATs, kubeconfig creds, HMAC keys) are write-only password inputs, sent once on create, never read back (create responses are masked). Resilience: a class `ErrorBoundary` wraps the routed outlet (reset on navigation via a location key), and a `NotFound` route replaces `<Navigate to="/">`.

**Tech Stack:** React + TS + Vite + Tailwind (Nocturne tokens) + TanStack Query + react-router v6 + Vitest/RTL/MSW v2. Kit primitives in `web/src/ui/`.

## Invariants (apply to EVERY task)

- **Design tokens only.** No hex, no `dark:` variants, no raw palette (`gray-`/`blue-`/etc.), no `text-brand-deep`. Compose from Nocturne tokens + kit primitives (`Sheet`, `Button`, `Input`, `Textarea`, `Select`, `Pill`, `ConfirmDialog`, `EmptyState`, `Card`, `cn`). Both light + dark render (`npm run smoke`).
- **SECURITY — the create forms carry credentials; enforce:**
  - Secret fields — rotation `admin_dsn`/`hmac_key`/`notify_hmac_key`, sync `pat`/`ca_cert`/`token`, dynamic `admin_dsn` — are `<Input type="password" autoComplete="off">`. They are held only in ephemeral create-form state, sent once in the POST body, and **NEVER** rendered from a fetched value (the list + create responses are masked — they contain no secrets). Reset the form (secrets → `''`) when the Sheet opens AND after a successful create AND on cancel/close.
  - No secret value in any toast, log, or error string. On create error, surface only `errorMessage(e)` (the server's curated 400/409 message), never echo the payload.
  - The `ConfigPicker` lists config **names/paths only** (project / env / config) — never secret values. It reuses `useProjectConfigMap`, which is already 403-tolerant.
  - Create responses are masked (no `admin_dsn`/`pat`/`token`/etc.) — do not attempt to display any credential after creation.
- **403-tolerance:** creating needs `rotation:manage`/`sync:manage`/`dynamic:manage`. A 403 on POST surfaces as an inline `errorMessage` + danger toast (the panel's list already self-gates 403 via `useFanOut`). Do not pre-hide the New button (no cheap cap probe) — mirror the Instance-section pattern from N5.
- **Mock-drift rule:** msw mocks in tests mirror the Go wire shapes EXACTLY (field names, nesting, status codes, error `code`s). See "Wire shapes" below.
- **No behavior regressions:** the existing ops actions (rotate/sync/pause/interval/delete/issue/renew/revoke) and their tests stay untouched and green.

## Wire shapes (verified against `internal/api/*_handlers.go` — mirror EXACTLY)

Secrets live in the nested `config`/`creds` objects (all `omitempty`). Every create returns **201 with a masked view — no secrets**.

```ts
// POST /v1/rotation/policies  (cap rotation:manage) → 201 rotationView (masked)
{ config_id, secret_key, type: 'postgres'|'webhook', interval_seconds: number,
  config: {
    // postgres:
    admin_dsn,          // SECRET
    role, password_len: number,
    // webhook:
    url, hmac_key,      // hmac_key SECRET
    // optional notify (either type):
    notify_url, notify_hmac_key   // notify_hmac_key SECRET
  } }
// 409 "conflict" if a policy already exists for this config+key.

// POST /v1/sync/targets  (cap sync:manage) → 201 syncView (masked)
{ config_id, provider: 'github'|'k8s', prune?: boolean /* nil→true */, interval_seconds: number,
  addr: { owner, repo, environment?     // github
        , namespace, secret_name },      // k8s
  creds: { pat, api_url?                 // github: pat SECRET
         , api_url, ca_cert, token } }    // k8s: ca_cert + token SECRET
// NOTE provider discriminator is 'k8s' (NOT 'kubernetes').

// POST /v1/dynamic/roles  (cap dynamic:manage) → 201 roleViewJSON (masked)
{ config_id, name, default_ttl_seconds: number, max_ttl_seconds: number,
  config: {
    admin_dsn,               // SECRET
    creation_statements,     // SQL template: {{name}} {{password}} {{expiration}}
    revocation_statements?,  // SQL template (optional)
    renew_statements?        // SQL template (optional)
  } }
// 409 if a role with that name already exists for the config.

// Error envelope (all): { "error": { "code": "...", "message": "..." } }
```

```tsx
import { useProjectConfigMap, ProjectFilter } from './useAggregated'
import { Select } from '../ui/Select'

export function ConfigPicker({ filter, value, onChange, label = 'Config' }: {
  filter: ProjectFilter; value: string; onChange: (id: string) => void; label?: string
}) {
  const { map, isLoading } = useProjectConfigMap(filter)
  const opts = [...map.values()]
    .map((c) => ({ id: c.configId, label: `${c.projectName} / ${c.envName} / ${c.configName}` }))
    .sort((a, b) => a.label.localeCompare(b.label))
  return (
    <Select label={label} value={value} onChange={(e) => onChange(e.target.value)}>
      <option value="">{isLoading ? 'Loading…' : opts.length ? 'Select a config…' : 'No configs available'}</option>
      {opts.map((o) => <option key={o.id} value={o.id}>{o.label}</option>)}
    </Select>
  )
}
```

```tsx
import { Component, ReactNode } from 'react'
import { Button } from '../ui/Button'

function digest(msg: string): string {
  let h = 0
  for (let i = 0; i < msg.length; i++) h = (h * 31 + msg.charCodeAt(i)) | 0
  return (h >>> 0).toString(36).slice(0, 8)
}

export class ErrorBoundary extends Component<{ children: ReactNode }, { error: Error | null }> {
  state = { error: null as Error | null }
  static getDerivedStateFromError(error: Error) { return { error } }
  componentDidCatch(error: Error) { console.error('UI crash:', error) }
  render() {
    const { error } = this.state
    if (!error) return this.props.children
    const id = digest(`${error.name}:${error.message}`)
    return (
      <div className="mx-auto mt-16 flex max-w-md flex-col items-center gap-3 rounded-card border border-line bg-surface-2 p-6 text-center shadow-elev-1">
        <p className="text-[15px] font-semibold text-ink">Something broke</p>
        <p className="text-[12.5px] text-ink-mute">An unexpected error occurred rendering this page. Reference <span className="font-mono text-ink-body">{id}</span>.</p>
        <div className="mt-1 flex gap-2">
          <Button variant="primary" onClick={() => window.location.reload()}>Reload</Button>
          <Button variant="secondary" onClick={() => void navigator.clipboard?.writeText(`[${id}] ${error.name}: ${error.message}\n${error.stack ?? ''}`)}>Copy details</Button>
        </div>
      </div>
    )
  }
}
```
