The eight changes that carry the redesign

1Design tokens first

Palette, type scale, radius, and shadows wired into Tailwind theme config. Components reference roles (brand, surface, muted) — never raw gray-400. Dark mode falls out nearly free.

2Surfaces & depth

Tinted page background, white cards with hairline border + soft shadow. Content sits on the page instead of bleeding white-into-white.

3Brand mark + top bar

Geometric split-hexagon mark (two faces of Janus: encrypt/decrypt), breadcrumb, seal-status pill, and user avatar menu replacing two loose outlined buttons.

4Env color coding

Doppler's signature: dev blue, staging amber, prod red — repeated in the sidebar, the config header, and any destructive confirm. Prod always feels different.

5Editor as a real grid

Card table, sticky header, hover-revealed icon actions with tooltips, origin pills, diff rails (green/amber/red edge) on pending rows, and the dirty-state bar with Save as vN.

6Mono = secret

ui-monospace exclusively for keys and values; sans for all chrome. The texture contrast makes secret material instantly scannable — it's the product's typographic signature.

7Feedback surfaces

Toasts for save/copy/errors, skeleton loaders instead of “Loading…”, styled dialogs instead of bare inline forms. Every action visibly lands.

8No more blank pages

Branded landing with “Create your first project” CTA; every empty list gets an icon + one-liner + action. A lone gray sentence never ships again.
