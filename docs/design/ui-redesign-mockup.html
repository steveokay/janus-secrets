Visual redesign · source of truth

Janus — self-hosted secrets, dark-first

A calibrated near-black redesign for the Janus SPA: command-palette navigation, environment-column project boards, and a batched secret editor. Dark is primary; toggle the control top-right to preview light. Every color and type decision derives from the token system documented at the bottom.
